}

ÉTAPE 2 : Note la réponse et garde le même session_id

bash

ÉTAPE 4 : Demande ton nom avec le MÊME session_id
json
{
  "query": "Quel est mon nom ?",
  "session_id": "ma_session_unique_123"   ← MÊME ID !
}
Ce que tu as fait (incorrect pour tester la persistance)
Ce que tu as envoyé	Problème
"session_id": "string"	C'est une nouvelle session à chaque fois
Session différente du premier test	N'a pas accès à l'historique de l'ancienne
