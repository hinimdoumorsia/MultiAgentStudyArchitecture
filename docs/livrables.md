# 12. Livrables

Dans le cadre de ce projet de recherche, les livrables suivants ont été réalisés et sont mis à disposition afin d’assurer la reproductibilité et la démonstration du système proposé.

- **Article scientifique**  
  Le document complet présentant la méthodologie, les expériences et les résultats du projet est disponible ici :  
  [Article scientifique (PDF)](https://drive.google.com/file/d/1g6c-XtSc2BTC71z2XeQZjtR5w92EpYwA/view?usp=sharing)

- **Code orchestrateur (système multi-agent + routeur dynamique)**  
  L’implémentation complète du système est disponible sur GitHub :  
  [Accéder au dépôt GitHub](https://github.com/hinimdoumorsia/MultiAgentStudyArchitecture)

- **Schémas d’architecture**  
  Les schémas d’architecture (hiérarchique, distribuée et pipeline de routage) sont inclus dans le dépôt GitHub dans la documentation du projet.

- **Interface de démonstration**  
  Une interface de démonstration est intégrée au projet permettant de tester le routeur dynamique et de visualiser la sélection automatique de l’architecture selon la requête utilisateur.
